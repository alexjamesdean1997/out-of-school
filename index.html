Twitter

Twitter est le premier outils que j'utilise pour ma veille, c'est sans doute la plateforme qui réunit la plus grande communauté de dev dans le monde. Il m'arrive fréquement de tomber sur des articles et des projets intéressant en consultant mon feed Twitter. Voici une petite liste de compte Twitter que je conseillerai à tout développeur web de suivre.

La Ferme du Web

La Ferme du Web est un compte dédiée au partage des connaissances, astuces, outils et bonnes pratiques du web / mobile. Les thématiques sont variées et s'adressent particulièrement aux développeurs, webdesigners, webmarketeurs, chefs de projets, webmasters et tous curieux du web.

Tweets by LaFermeDuWeb

CodePen.io

CodePen est une référence dans le domaine du développement d'interface web, il s'agit d'un environnement de développement social pour les concepteurs et les développeurs front-end. La plateforme permet de montrer son travail, créez des cas de test pour apprendre, déboguer et trouver de l'inspiration. Je suis le compte tweeter de codePen pour enregistrer des Pens qui pourront me servir sur des futurs projets

Tweets by CodePen

CSS Tricks

Créé et maintenu par Chris Coyier, une personnalité connue dans le domaine du web-design / developpement web, CSS Tricks est un site qui publie des article sur le développement front, le plus souvent sur des nouveautés CSS. Suivre le compte twitter de CSS Tricks est un bon moyen de rester à la page.

Tweets by CSS Tricks

Changelog

Changelog s'adresse à tout les développeurs en quête d'actualité dans domaine du web. Ce compte propose certains des meilleurs podcasts pour les développeurs et les nouvelles importantes quotidiennement.
